Element defaults

We don't reset base tags, we set them. For good typography, just use good markup. The approprite semantic heading level, for example, should match up with the appropriate visual style.

All block level content tags, like <p> have the same margin-bottom. This ensures good vertical flow, and also gives us a standard for our content containers. The docSection content container for example, has padding on all sides but the bottom (because the margin on the last block element provides the whitespace on the bottom).

For a full example of element defaults see the Sassquatch type specimen.
