Pill Labels

Standard labels for member titles and/or status. Using an inline tag like <span> is recommended.

Organizer 
    <span class="pill-label--leader">Organizer</span>

Label Types

Class Example
pill-label Generic label
pill-label--hollow Generic hollow label
pill-label--friend Friend indicator
pill-label--leader Role indicator
pill-label--paid Dues paid indicator
pill-label--unpaid Dues unpaid indicator

Tearsheet

Tearsheets act as a universal companion icon to upcoming Meetups. They contain the date of the event, but should never be relied on as the only conveyance of the event date.

Tile Grid

For a grid-like collection of items. You can control the number of columns with class modifiers. By default, tile content is vertical aligned top. For equal height tiles, you must set a min-height in page-specific CSS.

This type of collection view is best placed over a background of $C_backgroundCollection.

Column Modifiers

You can set the default number of columns from 1 to 6 with a variant class:

Class Description
tileGrid tileGrid--1col single column of cards
tileGrid tileGrid--2col two columns of cards
tileGrid tileGrid--3col three columns of cards
tileGrid tileGrid--4col you get the idea

Media Query Modifiers

tileGrid also supports media query column modifiers. You can declare a specific number of columns for:

  • 1-6 columns at the "medium" breakpoint
  • 1-6 columns at the "small" breakpoint

Examples:

Example Description
tileGrid tileGrid_atSmall--1col snap to 1 column at "small" breakpoint
tileGrid tileGrid_atMedium--3col snap to 3 columns at "medium" breakpoint
tileGrid tileGrid--6col tileGrid_atMedium--3col tilegrid_atSmall--1col 6 columns by default, 3 columns at medium, 1 column at small
